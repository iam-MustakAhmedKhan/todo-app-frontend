
# Simple Todo app

Todo application frontend




## Installation

To install this project in you local machine you have to clone this repository first


```bash
  https://github.com/iam-MustakAhmedKhan/todo-app-frontend.git my-project
  cd my-project
  npm install
  npm run dev
```
